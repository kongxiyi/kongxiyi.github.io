node版本控制

发表于2022-03-20|更新于2022-06-24|node
|阅读量:

环境准备

使用工具 nvm

  1. 官网下载：https://github.com/coreybutler/nvm-windows/releases
  2. 选择 nvm-setup.zip
  3. 解压，点击 exe，一路无脑安装
  4. 打开 nvm 文件夹下的 setting.txt,加入下面两句话

使用

  1. 输入命令行 nvm ls anaillable 查看可用的 node.js 版本号
  2. nvm install 版本号(例如：nvm install 12.17.0)
  3. nvm use 版本号（例如：nvm use 12.17.0）即可选择你本地所使用的 Node.js 版本
  4. 输入命令行 nvm ls 查看你安装的所有 node.js 版本号，以及你当前所选择的 node.js 运行版本

注意事项

使用 nvm 装的 node 默认不会配置 npm，也就是说会出现 node -v 成功，但是 npm -v 不成功的现象

解决方式：

  1. 查看 node 版本号，如 12.17.0
  2. 到 node 官网，查看对应 npm 版本 https://nodejs.org/zh-cn/download/releases/
  3. 到 npm 官网 https://npm.taobao.org/mirrors/npm/，下载对应版本号
  4. 下载完成，将解压后的文件复制到 nvm/v12.17.0/node_modules 下，并且重新命名为 npm（必须命名为 npm，此处以 12.17.0 版本举例）
  5. 将 nvm/v12.17.0/node_modules/npm/bin 文件夹下的四个文件移动到 nvm/v12.17.0 下
  6. cmd，使用 npm -v 进行验证

参考博客：https://blog.csdn.net/fenfeidexiatian/article/details/96993384

可能遇到的问题

解决：此时是因为无权限，使用管理员身份打开命令行管理器即可。
