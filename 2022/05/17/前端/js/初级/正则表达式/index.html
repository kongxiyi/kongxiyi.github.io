正则表达式

发表于2022-05-17|更新于2022-06-12|js基础
|阅读量:

限定符

  • ?: 0 或者 n 次
  • *: 0 或者 n 次
  • +： 1 或者 n 次
  • {n, m}
    • 可以只输入一个数字，如 { n }，表示只有 n 次
    • 可是输入 { n, }，表示 n 到无穷次

| 运算符

[] 与 ()

  • [] 表示取值定义域
  • () 仅表示一个整体

元字符

元字符 意义
\d 数字字符，等同于[0,9]
\w 英文字符 + 数字 + 下划线
\s 空白字符，包括 tab+换行
. 任意字符，但不包含换行符
^ 行首
$ 行尾

其他的，如\D，\w，\s 都是非\？的意思，如\D 表示非数字字符

贪婪/懒惰匹配

<.+> 默认贪婪匹配任意字符
<.+?> 懒惰匹配任意字符

API

regexObj.exec(str)

test

match

regexpSymbol.match

对正则表达式匹配字符串时，@@match方法用于获取匹配结果，这个方法的使用方式和 String.prototype.match()相同，不同之处是 this 和参数顺序。

matchAll

regexpSymbol.matchAll

[@@matchAll]方法返回对字符串使用正则表达式的所有匹配项，这个方法的使用方式和 String.prototype.matchAll()相同，不同之处是 this 和参数顺序。

replace

regexp[Symbol.replace](str, newSubStr|function)

@@replace方法会在一个字符串中用给定的替换器，替换所有符合正则模式的匹配项，并返回替换后的新字符串结果，用来替换的参数可以是一个字符串或是一个针对每次匹配的回调函数，这个方法基本可以和 String.prototype.replace()一样使用，不同之处是 this 和参数顺序。

search

regexpSymbol.search @@search方法执行了一个在给定字符串中的一个搜索以取得匹配正则模式的项，这个方法的使用方式和 String.prototype.search()相同，不同之处是 this 和参数顺序。

split

@@split方法切割 String 对象为一个其子字符串的数组，这个方法的使用方式和 String.prototype.split()相同，不同之处是 this 和参数顺序。
